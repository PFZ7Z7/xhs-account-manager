## 技术栈

- **语言**: Python 3.10+
- **HTTP**: requests / httpx
- **存储**: SQLite
- **CLI**: click / typer
- **配置**: pydantic + yaml
- **测试**: pytest
- **CI**: GitHub Actions

---

## 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 签名算法变更 | 高 | 模块化设计，易于更新 |
| Cookie 过期 | 中 | 自动检测，提示更新 |
| API 限流 | 中 | 请求队列，速率控制 |
| 账号风控 | 高 | 模拟人类行为，随机延迟 |
